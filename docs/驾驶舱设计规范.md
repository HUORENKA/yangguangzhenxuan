# 国采e购商城 · 运营驾驶舱设计规范

> 面向运营部门负责人的大屏驾驶舱，用于快速掌握平台核心业务数据与合规情况。

---

## 一、设计风格

### 1.1 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景主色 | `#0a1628` ~ `#0d1b2a` | 深蓝偏黑，营造专业、沉稳氛围 |
| 背景辅色 | `#1b2838` | 模块背景，略浅于主背景 |
| 主强调色 | `#00d4ff` / `#4fc3f7` | 亮蓝/天蓝，用于数据、标题、边框 |
| 次强调色 | `#00e5ff` | 青色，用于高亮、发光效果 |
| 数据色 | `#ffd54f` / `#ffb74d` | 黄/橙色，用于关键指标、图例 |
| 文字主色 | `#e0e7ef` | 浅灰白，正文内容 |
| 文字辅色 | `#8b9cb5` | 灰色，次要信息 |

### 1.2 视觉元素

- **光影效果**：标题左侧竖条、模块边框、数据卡片描边采用发光效果（`box-shadow`、`text-shadow`）
- **几何图形**：圆角矩形、六边形图标、环形底座，统一科技感
- **纹理与细节**：背景可带轻微网格或点状纹理，提升精细度
- **字体**：无衬线字体（如思源黑体、PingFang SC），标题加粗，数据字号加大

### 1.3 整体感受

- 高端、智能、数据驱动
- 强调数据的实时性和洞察力
- 适合运营部门负责人快速掌握核心业务状态

## 二、大屏驾驶舱内容

### 2.1 顶栏

| 元素 | 说明 |
|------|------|
| 主标题 | 「国采e购商城运营驾驶舱」或「阳光优采合规分析」 |
| 时间筛选 | 累计 / 本月 / 本年 / 自定义 |
| 日期范围 | 如：2025-05-01 ~ 2025-05-31 |
| 分站维度 | 全平台 / 指定区域分站 / 指定企业分站 |

### 2.2 左侧模块：项目数据与合规趋势

| 模块 | 内容 | 图表类型 |
|------|------|----------|
| **近一年采购成交趋势** | 成交订单数（柱状图，天蓝）+ 成交订单金额（折线图，黄色） | 组合图 |
| **近一年采购合规趋势** | 合规订单数（柱状图，橙/黄）+ 合规订单金额（折线图，浅蓝） | 组合图 |

### 2.3 中间模块：核心指标与洞察

| 模块 | 内容 |
|------|------|
| **核心业务总览（顶部）** | 合规订单数量、合规订单总金额、合规订单数量占比（三块大数字卡片） |
| **阳光采购中心（中部）** | 以「阳光采购」或「国采e购」为核心，环绕展示：累计成交订单、累计成交金额、今日成交订单、今日成交金额、上架商品总数、供应商总数、商品类目总数 |
| **采购商品类目 TOP5** | 柱状图展示采购金额最高的 5 个类目（如办公用纸、办公耗材、办公文具、电脑及配件、办公电器） |

### 2.4 右侧模块：合规明细

| 模块 | 内容 |
|------|------|
| **采购单位合规情况** | 列表 + 水平进度条，展示各采购单位合规率（如 100%、99.5%、98.9%） |
| **供应商合规情况** | 列表 + 水平进度条，展示各供应商合规率（如 92%、91.1%、89.8%） |

### 2.5 可选扩展

- **运营待办**：待审核企业认证、待审核供应商认证、待审核采购单、异常订单
- **分站订单占比**：总站 / 区域分站 / 企业分站 的订单量、金额占比（饼图）

## 四、图表规范

### 4.1 近一年采购成交趋势

- **类型**：柱状图 + 折线图（双 Y 轴）
- **X 轴**：月份（如 24-06 ~ 25-06）
- **左 Y 轴**：成交订单数（万个）
- **右 Y 轴**：成交订单金额（万元）
- **配色**：柱状图天蓝渐变，折线图亮黄
- **标题**：左侧黄色竖条 + 白色发光字体

### 4.2 近一年采购合规趋势

- **类型**：柱状图 + 折线图（双 Y 轴）
- **X 轴**：月份
- **左 Y 轴**：合规订单数（万个）
- **右 Y 轴**：合规订单金额（万元）
- **配色**：柱状图橙/黄，折线图浅蓝
- **标题**：左侧黄色竖条 + 白色发光字体

### 4.3 采购商品类目 TOP5

- **类型**：垂直柱状图
- **配色**：蓝-青发光渐变柱体，环形底座
- **数据标签**：柱顶显示金额（如 16.48 亿元），柱底显示类目名称
- **标题**：左侧黄色竖条 + 主标题「采购商品类目TOP5」+ 副标题「销售金额」

### 4.4 合规情况列表（采购单位 / 供应商）

- **布局**：排名图标 + 名称 + 进度条 + 百分比
- **排名图标**：六边形或圆形，金色/橙色背景
- **进度条**：蓝/青渐变，水平填充
- **标题**：左侧黄色竖条 + 白色/浅蓝字体

## 六、技术实现建议

| 项目 | 建议 |
|------|------|
| 图表库 | ECharts（支持组合图、发光效果、深色主题） |
| 布局 | CSS Grid / Flexbox，适配 16:9 大屏 |
| 数据 | 复用 `mock.js` 结构，扩展合规、类目、单位/供应商等字段 |
| 动画 | 数字滚动、图表入场动画，增强视觉冲击力 |
